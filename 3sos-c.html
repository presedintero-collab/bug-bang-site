3SOS-C

3SOS-C este un model ternar diferențial software-first în care starea centrală 0 nu este lipsă, ci echilibru activ între două ramuri complementare.

Ce este și ce nu este

3SOS-C nu este prezentat ca procesor ternar fizic validat și nu revendică superioritate universală față de calculul binar. Este un formalism ternar diferențial, bibliotecă Python, DSL, simulator și direcție de explorare hardware.

T = {-1, 0, +1}
0 = echilibru diferențial

Regula diferențială

Modelul folosește două ramuri complementare, N și P. Ieșirea este determinată de diferența dintre ele, urmată de cuantizare ternară.

D = N - P
CondițieIeșireSens
D > 0Y = -1dominantă negativă
D = 0Y = 0echilibru
D < 0Y = +1dominantă pozitivă
